# 網頁網址前端程式網頁設計：期末成果報告

## 專案名稱：課程作業彙整平台

> **專案總覽**：本專案為一個「單頁式整合網頁應用程式 (SPA)」，核心目的在於系統化管理、展示並解說 2026 學年度全學期的網頁前端與 JavaScript 全端開發成果。介面採用現代化 Flexbox 與 Grid 響應式佈局，並透過 **非同步 JavaScript (Fetch API)** 實時串接遠端 GitHub 儲存庫（Repository），達成免重新整理、即時預覽程式代碼與題目解析的高互動性設計。

## 🛠️ 各階段作業實作內容述要

本平台完美收錄、彙整了本學期從「靜態網頁視覺排版」逐步邁向「高階非同步全端程式邏輯」的完整成長軌跡：

### 1. 靜態網頁基礎與視覺佈局

* **作業一：個人網頁 (aboutme.html)**
* **核心技術**：HTML5 語意化標籤、CSS3 卡片化佈局（Card Design）、邊框圓角與陰影效果。
* **內文概述**：以精簡俐落的視覺美學，呈現個人簡介、家鄉背景（嘉義）、就讀學校（國立金門大學資訊工程學系）以及未來職涯目標，為後續複雜網頁結構進行熱身。


* **作業二：表單設計 (問卷調查表)**
* **核心技術**：HTML `<form>` 元素控制、`required` 屬性欄位驗證、CSS 緞帶（Ribbon）視覺效果設計。
* **內文概述**：實作全功能的會員問卷，整合下拉式選單、單選鈕與複選框。同時透過 JavaScript 優化使用者體驗（UX），達成「年月日連動生成選單」與「勾選其他時動態啟用文字輸入框」之高互動邏輯。



---

### 2. JavaScript 程式核心與演算法思維

* **作業三：Hello JS (主控台入門)**
* **核心技術**：JavaScript 執行環境、開發者工具（DevTools）除錯、`console.log()` 輸出。
* **內文概述**：初探網頁程式的動態生命週期，演練瀏覽器主控台的基礎輸出，建立基本的程式監聽概念。


* **作業四：JavaScript 基礎練習 (共10題)**
* **核心技術**：`if-else` 流程控制、巢狀迴圈（Nested Loops）、JSON 字串解析、基本資料結構運算。
* **內文概述**：挑戰 10 道演算法題庫，包含 BMI 計算器、經典數字金字塔、倒數計時、攝氏/華氏溫度轉換、購物車總金額加總及通訊錄搜尋系統，厚植核心邏輯思維。



---

### 3. 全端架構觀念與高階函式應用

* **作業五：OpenCode 網站專案開發**
* **核心技術**：Node.js 與 Express 後端視圖整合、模組化網頁版面配置、Markdown 開發筆記管理。
* **內文概述**：跨入全端工程領域，本專案彙整了「專案記憶開發筆記」與「部落格開發摘要」，記錄了路由（Routing）配置、中間件應用與模組化版面的整合歷程。


* **作業六：JavaScript 函式與參數進階練習 (共10題)**
* **核心技術**：回呼函式 (Callback)、箭頭函式 (Arrow Functions)、立即執行函式 (IIFE)、高階函式 (HOF)、參數傳值與傳址陷阱。
* **內文概述**：全面解構核心高階語法，探討閉包（Closure）與記憶體傳址（Pass by Reference）的陷阱，並手寫實作類似原生陣列 `.filter()` 與 `.map()` 邏輯的篩選器。


* **作業七：JavaScript 網誌邏輯演練 (共10題)**
* **核心技術**：物件解構賦值 (Destructuring)、樣板字串 (Template Literals) 動態拼接 HTML、模擬 SQL 資料庫查詢、Node.js 錯誤優先回呼模式 (Error-First Callbacks)。
* **內文概述**：將前端 UI 與後端資料庫資料進行銜接訓練。模擬網路請求、URL 路由參數解析（req.params.id）與錯誤捕獲機制，為全端網誌平台的資料流打通任督二脈。



---

### 4. 期中大型專案實作成果

* **期中作業：水果合成網頁遊戲**
* **核心技術**：2D 物理引擎（重力、碰撞體積）、Canvas 動態圖像堆疊、前端滑鼠/觸控事件監聽、GitHub Pages 靜態網頁自動部署。
* **內文概述**：融合這學期所學的物件導向與事件驅動程式設計，開發出一款具備流暢物理碰撞與物件「動態碰撞進化/升級」機制的休閒水果合成遊戲。

## 🚀 彙整平台技術亮點與亮眼功能

| 技術亮點 | 技術細節說明 |
| --- | --- |
| **單頁應用 (SPA) 體驗** | 精準使用 CSS 顯示切換（`display: block/none`），切換分頁時完全**免重新整理**，提供流暢的 App 般體驗。 |
| **動態 AJAX 代碼同步** | 利用原生 **Fetch API** 異步非同步讀取 GitHub 上的最新程式原始碼，不佔用本機靜態空間，資料永遠保持最新狀態。 |
| **即時題目與預期結果對照** | 內建互動式 JavaScript 資料庫，切換各小題（如 01.js、02.js）時，網頁會自動變更並呈現**該題目的目標與預期控制台輸出結果**，一目了然。 |
| **健全的錯誤處理機制** | 程式內含安全跨域與 Private 權限攔截排查指南（`.catch(error)`），當遠端連線異常時能優雅顯示排查警示，具備商用網頁的水準。 |
